Motion Controller Driver

The Motion Controller Driver implements the CIA402 profile and enables setting the drive status, operation mode and sending target values to the motion controller.

Services

Services

Type

Description

~/nmt_reset_node

Trigger

Resets CANopen Device the Proxy Device Node manages.

~/sdo_read

CORead

Reads an SDO object from the specified index, subindex and datatype of the remote device.

~/sdo_write

COWrite

Writes data to an SDO object on the specified index, subindex and datatype of the remote device.
